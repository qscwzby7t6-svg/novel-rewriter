# Aid Learning 在荣耀平板上的详细安装教程

> 零基础用户专用，一步一步复制粘贴操作

---

## 📋 安装前准备

### 确认设备信息

- **设备**: 荣耀平板（支持 HarmonyOS 或 Android）
- **系统版本**: HarmonyOS 3.0+ 或 Android 8.0+
- **存储空间**: 至少 2GB 可用空间
- **网络**: 稳定的 Wi-Fi 连接

---

## 第一步：关闭纯净模式（必须）

荣耀平板的纯净模式会阻止安装非应用市场的软件，必须先关闭。

### 操作步骤：

1. 打开 **设置**
2. 点击 **系统和更新**
3. 点击 **纯净模式**
4. 点击右上角 **⋮**（三个点）
5. 选择 **退出**
6. 点击 **仍然退出**
7. 输入锁屏密码确认

### 验证关闭成功：

返回设置主界面，再次进入 **系统和更新** → **纯净模式**，应该显示：

```
纯净模式
已关闭
```

---

## 第二步：下载 Aid Learning APK

### 方式 A：官方渠道下载（推荐）

1. 打开平板浏览器（推荐 Chrome 或华为浏览器）
2. 访问官网：https://www.aidlearning.net/
3. 点击 **"Download"** 或 **"下载"**
4. 选择 **Android 版本**
5. 等待下载完成

### 方式 B：GitHub 下载

1. 打开浏览器
2. 访问：https://github.com/aidlearning/AidLearning-FrameWork/releases
3. 找到最新版本（如 v0.88）
4. 点击 **Assets** 展开
5. 下载文件名类似 `aidlux-xxx.apk` 的文件

### 方式 C：国内镜像下载

如果官网下载慢，使用国内源：

1. 打开浏览器
2. 访问：https://www.pgyer.com/aidlearning
3. 点击 **下载**
4. 选择 **普通下载**

---

## 第三步：允许安装未知来源应用

### 操作步骤：

1. 打开 **设置**
2. 点击 **安全**
3. 点击 **更多安全设置**
4. 找到 **安装外部来源应用**
5. 点击 **浏览器**（或你使用的下载工具）
6. 打开 **允许安装应用** 开关

---

## 第四步：安装 Aid Learning

### 操作步骤：

1. 打开 **文件管理**
2. 进入 **下载** 文件夹
3. 找到下载的 `aidlux-xxx.apk` 文件
4. 点击 APK 文件
5. 点击 **安装**
6. 等待安装完成（约 1-2 分钟）
7. 点击 **打开**

## 第五步：初始配置

### 首次启动配置：

1. **欢迎界面**
   - 点击 **开始使用**
   - 或 **Start**

2. **权限申请**（全部允许）
   - 点击 **允许** 存储权限
   - 点击 **允许** 相机权限（可选）
   - 点击 **允许** 麦克风权限（可选）

3. **选择语言**
   - 选择 **简体中文**
   - 点击 **确定**

4. **初始化环境**（自动进行，约 3-5 分钟）
   ```
   正在初始化 Python 环境...
   正在下载依赖包...
   正在配置系统...
   ```
   等待进度条完成

5. **完成**
   - 看到主界面说明初始化成功

---

## 第六步：验证安装

### 验证 Python 环境：

1. 打开 Aid Learning 应用
2. 点击底部 **终端** 图标
3. 在终端中输入：

```bash
python --version
```

4. 应该显示：

```
Python 3.7.x
```

### 验证 pip 可用：

```bash
pip --version
```

应该显示 pip 版本信息。

---

## 第七步：安装项目依赖

### 操作步骤：

1. 在 Aid Learning 中打开 **终端**
2. 创建项目目录：

```bash
mkdir -p ~/projects
cd ~/projects
```

3. 下载项目代码：

```bash
# 方式1: 使用 git
git clone https://github.com/qscwzby7t6-svg/novel-rewriter.git

# 方式2: 如果没有 git，手动下载 ZIP 解压
```

4. 进入项目目录：

```bash
cd novel-rewriter
```

5. 安装依赖：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

⚠️ **注意**: 安装过程可能需要 10-20 分钟，请耐心等待。

---

## 第八步：运行项目

### 启动服务：

1. 在 Aid Learning 终端中：

```bash
cd ~/projects/novel-rewriter
python -m uvicorn backend.main:app --host 0.0.0.0 --port 8000
```

2. 看到以下信息说明启动成功：

```
INFO:     Uvicorn running on http://0.0.0.0:8000
```

### 访问系统：

1. 打开平板浏览器
2. 访问：

```
http://localhost:8000/docs
```

3. 看到 API 文档页面说明运行成功！

## 常见问题解决

### 问题 1：安装时提示 "安装失败" 或 "解析错误"

**原因**: APK 文件下载不完整或损坏

**解决**:
1. 删除已下载的 APK
2. 重新下载
3. 确保下载过程中网络稳定

---

### 问题 2：初始化时卡住不动

**原因**: 网络问题导致依赖下载失败

**解决**:
1. 检查 Wi-Fi 连接
2. 切换网络重试
3. 或手动配置 pip 镜像：

```bash
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

---

### 问题 3：提示 "存储空间不足"

**原因**: 平板可用空间不足

**解决**:
1. 清理不必要的应用和文件
2. 确保至少有 2GB 可用空间
3. 或卸载 Aid Learning 后重新安装到 SD 卡（如果支持）

---

### 问题 4：安装后闪退

**原因**: HarmonyOS 兼容性问题

**解决**:
1. 确保已关闭纯净模式
2. 尝试安装旧版本 Aid Learning
3. 或联系 Aid Learning 官方支持

---

### 问题 5：pip 安装依赖时内存不足

**原因**: 平板内存有限

**解决**:

```bash
# 使用预编译包，减少内存占用
pip install -r requirements.txt --only-binary :all: -i https://pypi.tuna.tsinghua.edu.cn/simple
```

---

### 问题 6：无法访问 localhost:8000

**原因**: Aid Learning 网络配置问题

**解决**:
1. 尝试使用 `127.0.0.1` 代替 `localhost`：

```
http://127.0.0.1:8000/docs
```

2. 或在 Aid Learning 中打开内置浏览器访问
